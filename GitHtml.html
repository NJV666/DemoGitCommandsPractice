Git Introduction

  • Git was developed by Linus Torvalds in 2005 for Linux kernel development.
  • Git is a version control system (VCS) or Source code management (SCM).
  • Git is a distributed version control system (DVCS) widely used for tracking changes in source code during software development.
  • It is a platform independent.
  • It is free and open source.
  • Git can handle large projects efficiently.
